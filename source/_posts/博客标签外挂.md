Hexo + Butterfly 主题内置了很多「标签外挂」（Tag Plugins），可以很方便地写出更美观、可读性更高的文章效果。  
这篇文章主要整理我自己在写博客时**最常用的几个标签外挂**，方便以后查阅，也希望能帮到同样在使用 Butterfly 的你。

本文主要包括：

- **提示块标签 `note`**
- **按钮标签 `btn`**

---

## 提示块标签：`note`

提示块标签常用于**突出一段说明、注意事项或补充内容**。  
基本语法如下：

{% note [color] [style] %}
这里是提示块内容
{% endnote %}- **`color`（可选）**：颜色 / 类型，例如：`default`、`primary`、`success`、`info`、`warning`、`danger`
- **`style`（可选）**：展示风格，例如：`simple`、`modern`、`flat`、`disabled`、`no-icon`

如果不写参数，则使用主题默认样式。

## 按钮标签：`btn`

按钮标签可以在文章中插入一个**带样式的按钮链接**，常用来引导读者访问文档、Demo 或外部站点。

基本语法如下：

{% btn '链接地址', 按钮文字, 图标类名, 颜色 尺寸 %}- **链接地址**：使用单引号包裹，例如 `'https://butterfly.js.org/'`
- **按钮文字**：显示在按钮上的文本
- **图标类名**：一般是 Font Awesome 的类名，例如：`far fa-hand-point-right`
- **颜色**（可选）：如 `blue`、`pink`、`red`、`purple`、`orange`、`green` 等
- **尺寸**（可选）：如 `larger` 等

> 颜色和尺寸放在最后，用空格分隔，顺序一般为：**颜色 + 尺寸**。

### 常见按钮示例

下面以 Butterfly 官方文档为例：

{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,blue larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,pink larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,red larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,purple larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,orange larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,green larger %}

对应写法如下：

{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,blue larger %}
{% btn 'https://butterfly.js.org/',Butterfly,far fa-hand-point-right,pink larger %}

## 小结

- **提示块 `note`**：用于强调段落内容，可以通过颜色（`default`、`primary`、`success` 等）和风格（`simple`、`modern`、`flat`、`disabled`、`no-icon`）进行组合搭配。
- **按钮 `btn`**：用于在文章中插入更醒目的链接按钮，支持设置图标、颜色和大小。

合理使用这些标签外挂，可以让你的技术文章**结构更清晰、重点更突出、交互更友好**。如果想了解更多标签外挂，可以前往 Butterfly 官方文档查看更多示例和高级用法。
